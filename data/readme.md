### 使うフォーマット
通貨単位は ISO 4217 を使用。
言語の表記は ISO 639-1 を使用。
日付、時刻の表記は ISO 8601拡張表記 を使用。
紙サイズの表記は ISO 216 を使用。JIS B列(JIS P 0138)を使用する場合は**必ず**`JIS Bn`と表記すること。
