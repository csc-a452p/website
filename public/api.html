情報工学同好会(非公認) API (1.0)

Download OpenAPI specification:Download

このAPIは無保証で提供され、突然API仕様が変更される可能性があります。あらかじめご了承ください。
ほとんど静的ファイルを返してるだけです。

errata

正誤情報を取得する

正誤情報のリスト

正誤情報のある販売品のリストを返します。 errataIDが返り、errataIDから正誤情報の詳細を取得できます。

Responses

本番サーバ

https://csc.nikachu.net/data/errata.json
http://localhost:3000/data/errata.json

Response samples

  • 200
Content type
application/json
{
  • "errata": [
    • "ngo_vol1"
    ]
}

正誤情報のデータ

指定されたIDの正誤情報を返します

path Parameters
errataID
required
string
Example: ngo_vol1

販売品につくgoodsIDと同じになります。

Responses

本番サーバ

https://csc.nikachu.net/data/errata/{errataID}.json
http://localhost:3000/data/errata/{errataID}.json

Response samples

  • 200
Content type
application/json
{
  • "1": [
    • {
      • "at": "p.4 14,15行目(数式込)",
      • "wrongImg": "/assets/errata/ngo_vol1/wrong_1_1.svg",
      • "correctImg": "/assets/errata/ngo_vol1/correct_1_1.svg"
      }
    ]
}

goods

販売品に関する情報を取得する

販売品のリストを取得

全販売品のリストを取得します。
goodsIDの配列が返り、goodsIDから販売品の詳細を取得できます。

Responses

本番サーバ

https://csc.nikachu.net/data/goods.json
http://localhost:3000/data/goods.json

Response samples

  • 200
Content type
application/json
{
  • "goods": [
    • "ngo_vol1"
    ]
}

新刊や新商品である販売品を取得

新刊や新商品である販売品を取得します。 新刊の定義は一番近くに行ったイベントで初めて販売したものとします。

Responses

本番サーバ

https://csc.nikachu.net/data/goods_newly.json
http://localhost:3000/data/goods_newly.json

Response samples

  • 200
Content type
application/json
{
  • "newly": [
    • "ngo_vol1"
    ]
}

Responses

https://csc.nikachu.net/data/notes/list.json
http://localhost:3000/data/notes/list.json
